# Hydrus Client (web)

Минимальный быстрый веб-клиент для [Hydrus Network](https://hydrusnetwork.github.io/hydrus/),
работает напрямую с Client API. SolidJS + Vite + TypeScript, без бэкенда.

## Возможности

- Поиск файлов по тегам (`system:everything`, если поле пустое).
- Виртуализированная сетка миниатюр без сторонних либ — десятки тысяч файлов скроллятся плавно.
- Просмотр файла (изображения/видео) с тегами и метаданными.
- Настройки (адрес API + access key) хранятся в `localStorage`.

## Подготовка Hydrus

1. В Hydrus: **services → manage services → client api** — включить, запомнить порт (по умолчанию `45869`).
2. Создать **access key** с правами на поиск и просмотр файлов.
3. Включить **support CORS headers** (нужно для браузерного клиента с другого origin).

## Запуск

```bash
cp .env.example .env   # вписать адрес и ключ (необязательно)
npm install
npm run dev            # http://localhost:5173
```

Настройки можно задать тремя способами (приоритет сверху вниз):
`localStorage` (форма в UI) → `.env` (`VITE_HYDRUS_URL`, `VITE_HYDRUS_KEY`) → дефолт.
Если в `.env` заданы оба значения — приложение стартует сразу в сетке, без формы.

> `VITE_*` вшиваются в бандл при сборке, ключ окажется в JS. Нормально для
> локального использования; не публикуй такую сборку с реальным ключом.

### macOS / "Cannot find module @rollup/rollup-darwin-arm64"

[Баг npm с optional-зависимостями](https://github.com/npm/cli/issues/4828) — обычно
после переноса `node_modules` с другой платформы. Лечится переустановкой:

```bash
rm -rf node_modules package-lock.json && npm install
```

Сборка статики: `npm run build` → каталог `dist/` (отдаётся любым статик-сервером).

## Архитектура

- `src/api/hydrus.ts` — типизированный клиент API. JSON-запросы (поиск/метаданные) идут
  через `fetch` с ключом в заголовке; URL миниатюр/файлов для `<img>/<video>` несут ключ
  в query-параметре (на медиа-теги заголовки не повесить). URL стабильны по `file_id` →
  агрессивное кеширование браузером (контент иммутабелен).
- `src/components/VirtualGrid.tsx` — оконная виртуализация сетки фиксированного размера.
- `src/components/FileViewer.tsx` — модальный просмотр, метаданные тянутся лениво по клику.

## Куда расти

- Автокомплит тегов (`/add_tags/search_tags`) с дебаунсом.
- Префетч метаданных видимого окна, бесконечная подгрузка результатов.
- Сортировки (`file_sort_type`), фильтры, страницы Hydrus (`/manage_pages`).
- Редактирование тегов/рейтингов (`/add_tags/add_tags`).
- Кеш миниатюр в `Cache API`/IndexedDB для офлайн-повторов.
